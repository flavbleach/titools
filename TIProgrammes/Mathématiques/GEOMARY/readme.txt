GEOMARY est un formulaire exécutable pour les Première et Terminale S et ES sur les:
* suites arithmétiques
* suites géométriques
Les formules sont groupées par menu: définition, somme, variations, et limites.
Dans chaque menu, l'on peut basculer rapidement du mode arithmétique au mode géométrique, et ainsi comparer.


Compatibilité:
-------------

* La version <plus> est dans le dossier "83+ - 84+".
* La version <basic>  est dans le dossier "82Stats - 83 - 76.fr".

Moins il y a de modèles compatibles, plus la version sera performante sur ces modèles, car utilisant des fonctions spécifiques non disponibles sur les autres modèles.

Différences:
-----------

La version <plus>, permet sur TI-83+ et TI-84+ d'avoir plus de mémoire libre quand le programme n'est pas utilisé, en le stoquant en mémoire d'archive.
Quand le programme est utilisé, il est temporairement recopié en mémoire principale.


Installation:
------------

Il vous faut envoyer les fichiers suivants:

Version <basic>	| Version <plus>
----------------+----------------
		| GEOMARY
GEOMARY		| YGEOMARY	*
		| ZRESRCES

Les fichiers occupent environ:
- 2.2Ko en mémoire principale pour la version <basic>.
- 0.5Ko en mémoire principale + 2.2Ko en mémoire d'archive pour la version <plus>.
  (les fichiers accompagnés d'une (*) doivent être archivés - les autres doivent être désarchivés)

Le programme principal à lancer sur la calculatrice est GEOMARY.


Touches:
-------

* [MODE] [CLEAR]: quitte le programme PROPREMENT, en nettoyant la mémoire, réinitialisant la fenêtre graphique (sur laquelle on a dessiné le diagramme), et supprimant le partage d'écran
		  POUR EVITER TOUT PROBLEME, NE QUITTEZ PAS LE PROGRAMME AVEC [ON] MAIS AVEC CES TOUCHES QUI FERONT LE MENAGE
* [ENTER]	: retour à l'écran d'accueil
* [F1] - [F4]	: affiche la page correspondant au menu de bas d'écran
* [F5]		: bascule entre le mode "suite arithmétique" et le mode "suite géométrique"
